Sorting Algorithms Project

Using a template class to implement different input arrays, different sorting algorithms were implemented in order to compare swaps, compares, and completion times to compare to the theoretical Big O times.

Password Auditing Project Using Hashcat

Using password cracking tools such as John the Ripper and Hashcat, I was able to learn about auditing passwords by creating dictionaries given a company's password policy and different personal statements from each user. This taught me how to think efficiently to get each password in order to get them fast enough.

Operating Systems Scheduling Simulator

Programmed in C++, this project was an operating system scheduling simulator to compare four different algorithms: First Come First Serve, Priority First, Round Robin, and Shortest Run Time First

Buffer Overflow: Reverse Shell Project

After learning about binary memory exploitation techniques such as a buffer overflow, we were given a program vulnerable to this exploit. The requirement was to then use this program to create a buffer overflow payload to inject shellcode on the victim machine, connecting back to the attacker machine, bypassing the firewalls. This was a common reverse shell attack that companies must be aware of when setting policies for technology.
